Este es un programa en consola que imprime los primero 10 numeros primos

Para ejecutar el ejercicio puede:

Abrir una consola ubicada en la raiz del proyecto y ejecutar el archivo ejercicio1.jar
    java -jar ejercicio1.jar

o

Puede agregar el proyecto al IDE y ejecutarlo desde ahi como Java Aplication

Al ejecutarse se mostraran en consola la lista de resultado
